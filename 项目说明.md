# 汉字乐园 — 项目说明文档

## 一、项目介绍

**汉字乐园**（chinese-hanzi）是一款专为小学生设计的汉字与拼音学习 Web 应用。通过「源、玩、学、练」四个模块，将汉字起源、互动游戏、笔画学习与拼音练习融为一体，帮助孩子在轻松有趣的氛围中掌握汉字与拼音。

- **技术栈**：Python 3.13+、Streamlit、前端内联 HTML/CSS/JS  
- **运行方式**：单入口多页导航，侧栏统一绘制，页面切换流畅无整页刷新  
- **数据支撑**：内置汉字-拼音映射、学习字表（组词、例句、释义）及古典文化内容

---

## 二、项目特点

| 特点 | 说明 |
|------|------|
| **寓教于乐** | 结合历史起源、游戏闯关、笔画动画与填空练习，避免枯燥记忆 |
| **多感官学习** | 支持笔画动画、拼音/组词/例句朗读（浏览器 TTS）、古典背景音乐 |
| **规范书写** | 学字模块提供田字格与 Hanzi Writer 笔画动画，强调书写顺序与结构 |
| **即时反馈** | 游戏与练习具备正确/错误音效、连击提示、得分与进度展示 |
| **界面统一** | 全局背景与主题一致，隐藏状态栏与页脚，侧栏导航一次渲染、切换不闪屏 |
| **数据可扩展** | 字库与学习内容集中在 `data/dict.py`，便于增删字与例句 |

## 三、功能介绍

### 1. 首页（🏠）

- 展示项目标题「汉字乐园」与 Slogan「让学习汉字变得有趣！」
- 四个功能卡片：**源**、**玩**、**学**、**练**，点击可跳转对应页面
- 底部简要说明各模块（汉字起源、玩游戏、学汉字、练拼音），并提示可从左侧边栏切换功能

---

### 2. 源（📜）— 汉字起源与演变

- **古典风格页面**：仿古配色与背景纹理，营造「汉字由来」的氛围  
- **时间线展示**：按历史时期呈现汉字演变（如甲骨文、金文、小篆、隶书、楷书等），每段配有时代标签、时期与简短说明  
- **图文并茂**：支持古字形图片、示例汉字卡片，悬停有动效  
- **六书介绍**：象形、指事、会意、形声、转注、假借的卡片说明与示例  
- **趣味小知识**：以卡片形式展示与汉字或书写相关的小知识  
- **背景音乐**：可播放/暂停古典背景音乐（使用 OSS URL，无需本地 `static/music_bg.mp3`）

适合作为「汉字从哪来、怎么变」的启蒙与拓展阅读。

---

### 3. 玩（🎮）— 汉字消消乐

- **规则**：根据底部给出的**目标汉字**，在上方汉字卡片中点击与之**拼音相同**的字进行消除；错误点击会抖动提示。  
- **限时**：单局 30 秒倒计时，营造轻度紧张感。  
- **连击加分**：连续选对会显示连击数与额外加分，鼓励快速反应。  
- **状态栏**：显示剩余时间、得分、结束按钮；时间将尽时有警示动效。  
- **结果弹窗**：结束后展示得分、消除数量、正确率等，支持「再玩一次」。  
- **视觉与音效**：正确消除有缩放消失与飞字动画，错误有抖动与音效；可搭配简单粒子/彩带效果。

字与拼音来源于 `data/dict` 中的 `CHINESE_TO_PINYIN`，每局随机抽字，保证可玩性与复习面。

---

### 4. 学（📚）— 汉字学习

- **选字方式**：侧栏提供「随机」「下一个」及下拉框，可快速切换要学的字（字表来自 `WORD_TO_LEARN`）。  
- **田字格 + 笔画动画**：使用 Hanzi Writer 在田字格内书写当前汉字，支持**自动播放笔画顺序**与**重播**按钮，便于模仿规范笔顺。  
- **拼音**：展示该字拼音（含声调），可点击喇叭按钮发音（浏览器语音合成）。  
- **组词**：展示包含该字的词语，每个词语可单独发音。  
- **例句**：展示例句，每句可点击发音，帮助在语境中理解字义与用法。  
- **文字释义**：从 `WORD_TO_CONTENT` 等数据中展示释义表格（拼音、释义、出处、相关词等），适合家长或高年级学生拓展。

整体围绕「会写、会读、会用在词句里」设计，适合课后巩固与拓展。

---

### 5. 练（✏️）— 拼音与汉字练习

- **模式**：当前实现为**找拼音填句子**——给出带空白的句子，需从备选拼音中选出正确一项填入句中空缺。  
- **题目来源**：从 `WORD_TO_LEARN` 的例句中抽取带拼音标注的句子，将其中某字拼音挖空并生成若干干扰项。  
- **单轮设置**：每轮约 10 道题，正确 +10 分、错误 -5 分；支持计时、进度条与结束/继续练习。  
- **交互**：句子可朗读（TTS）；选对有空位填充动效与简单粒子反馈，选错有抖动与错误音效；结束后弹窗展示用时、得分、正确/错误数等。  

通过「在句子中选拼音」，强化字音与语境对应，适合在「学」之后做巩固练习。

## 四、项目结构概览

```
chinese-hanzi/
├── 汉字乐园.py          # 入口：Streamlit 多页导航、侧栏、全局样式
├── pages/
│   ├── 0_🏠_首页.py      # 首页与四模块入口
│   ├── 1_📜_源.py        # 汉字起源（时间线、六书、古风样式与背景音乐）
│   ├── 2_🎮_玩.py        # 汉字消消乐（限时、连击、结果统计）
│   ├── 3_📚_学.py        # 汉字学习（田字格、笔画、拼音、组词、例句、释义）
│   └── 4_✏️_练.py        # 拼音与汉字练习（找拼音填句子）
├── data/
│   └── dict.py           # 字表：CHINESE_TO_PINYIN、WORD_TO_LEARN、WORD_TO_CONTENT 等
├── static/                # 其他静态资源（背景音乐已改为 OSS URL）
├── pyproject.toml        # 项目依赖（Streamlit、pydub 等）
└── 项目说明.md           # 本说明文档
```

## 五、运行与依赖

- **Python**：>= 3.13  
- **主要依赖**：`streamlit`、`pydub`（见 `pyproject.toml`）  
- **运行**：在项目根目录执行 `streamlit run 汉字乐园.py`，在浏览器中打开提示的地址即可使用。
